# JobScrapper

채용사이트 스크래퍼

- BeautifulSoup 사용하여 채용사이트 크롤링 (Job Korea, Stack Overflow, We Work Remotely)
- 검색결과 csv파일로 저장
- 서버에 저장된 검색결과 불러오기
- AWS EC2 호스팅
